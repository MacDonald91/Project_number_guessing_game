# Guessing Number Game

Number Guessing Game

This is a simple number guessing game where you try to guess a random number chosen by the computer.

How to Play:

Run the script: Execute the Python script containing this code.
Enter a number: When prompted, type a positive integer as the highest possible number the computer can choose from.
Make guesses: The computer will then pick a secret number between 0 and the number you entered.
Guess the number: Follow the on-screen prompts and type your guesses as integers.
Win or lose: The game will continue until you guess the correct number.
Gameplay:

The computer will secretly choose a random number between 0 and the highest number you entered.
You will be prompted to guess the number.
After each guess, the game will provide feedback indicating if your guess is too high, too low, or correct.
The game keeps track of the number of guesses you make.
Objective:

Guess the secret number in the fewest attempts possible.

Additional Notes:

The game will validate your inputs to ensure you enter positive integers when prompted.
Enjoy the guessing game!
